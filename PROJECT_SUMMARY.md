# Vein 个人导航系统 - 项目总结

## 📊 项目概况

**Vein** 是一个现代化的个人导航系统，采用 Vue 3 + Vite 构建，支持三布局模式和毛玻璃 UI 设计。项目已完成核心功能开发和布局优化。

## 🎯 核心功能

### ✅ 已实现功能

#### 1. 数据驱动架构
- ✅ YAML配置文件支持 (`public/menu.yaml`, `public/settings.yaml`)
- ✅ 三级分类结构 (Categories → Groups → Links)
- ✅ 自动数据加载和解析

#### 2. 三布局模式
- ✅ **iOS模式**: 大图标网格布局，类似iOS桌面
- ✅ **列表模式**: 左侧毛玻璃侧边栏 + 右侧列表区域
- ✅ **网格模式**: 卡片式网格布局

#### 3. 交互功能
- ✅ 实时搜索功能 (标题、描述、URL)
- ✅ 一键复制链接 (悬停显示复制按钮)
- ✅ 布局模式切换
- ✅ 分类标签切换
- ✅ 设置侧边栏 (布局和主题设置)

#### 4. UI/UX设计
- ✅ 毛玻璃效果和背景模糊
- ✅ 平滑过渡动画
- ✅ 响应式设计
- ✅ 无Banner设计原则
- ✅ Favicon自动获取
- ✅ 全屏铺满布局
- ✅ 全局滚动条隐藏

## 🛠 技术栈

- **核心框架**: Vue 3 + Composition API
- **构建工具**: Vite
- **样式方案**: Tailwind CSS + PostCSS
- **图标库**: Lucide Vue Next
- **工具库**: @vueuse/core, js-yaml
- **包管理器**: pnpm

## 🎨 设计规范

### UI/UX规范
1. **无Banner原则**: 页面顶部直接展示搜索框
2. **毛玻璃效果**: 使用 `backdrop-blur-md` 和透明度效果
3. **精致过渡**: 所有交互都有 `transition-all duration-300`
4. **图标处理**: Google Favicon API + 首字母降级
5. **全屏布局**: 弹性布局确保内容区域全屏铺满
6. **隐藏滚动条**: 全局隐藏但保留滚动功能

### 布局模式规范

#### iOS模式
- 大尺寸图标网格排列
- 悬停时图标平滑放大 + 上浮效果 (`translate-y(-8px)`)
- 悬停时显示链接描述

#### 列表模式
- 左侧毛玻璃侧边栏
- 右侧紧凑链接列表
- 分组标题显示

#### 网格模式
- 卡片式网格布局
- 支持悬停效果
- 图标无背景纯净显示
